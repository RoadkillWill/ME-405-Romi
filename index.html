Project Overview

This project involves building and testing a Romi robot that follows a line track in a time-trial format.

Hardware & Wiring

We used sensors and actuators including:

  • Line-following IR sensors
  • Motor driver controlled by PWM
  • Microcontroller (e.g., STM32, Raspberry Pi Pico)

Software Implementation

We implemented a PID controller to optimize the robot's movement.


# Example PID control loop
error = setpoint - sensor_value
integral += error
derivative = error - previous_error
output = Kp * error + Ki * integral + Kd * derivative
